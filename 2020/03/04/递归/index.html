递归机制

什么叫递归？

直接或间接调用函数本身

递归调用规则
  1. 当程序执行到一个方法时，就会开辟一个 独立的空间（栈）
  2. 每个空间的数据（局部变量）是独立的
  3. 如果传入的是引用类型时（比如数组），就会共享该引用类型的数据
  4. 递归必须向退出递归的条件逼近，否则就会栈溢出

此时的结果时 只输出一次 n=2 当去掉else 时 输出结果时 n=2；n=3；n=4；

每次都会在栈里开一个*局部变量 *n ,输出时执行的第一个指令为n = 2;然后放回到上一个栈。

只有最顶级的栈会进入else 其他的在结束前就进入了 if 所以就不会有输出

递归能解决什么问题？

8皇后问题，汉诺塔，阶乘问题，迷宫问题。

快排，归并，二分查找，分治。

用栈解决的问题，用递归代码比较简洁
